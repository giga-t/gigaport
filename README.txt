========================================
  How to view this exported website
========================================

This website uses JavaScript modules that require
an HTTP server to work (they won't load from file://).

QUICK START:
  1. Make sure Node.js is installed (https://nodejs.org)
  2. Double-click "start.command" (macOS) or "start.bat" (Windows)
  3. The site will open in your browser at http://localhost:3333

ALTERNATIVE (if you have npx):
  npx serve .

The site will have full animations, scroll effects,
and interactive features — exactly like the original.
